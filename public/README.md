# Logo Instructions

## How to Add Your Logo

1. **Place your logo file** in this `public` folder
2. **Name it** `logo.png` (or `logo.svg`, `logo.jpg`, `logo.webp`)
3. **Recommended formats:**
   - **PNG** with transparent background (best for logos)
   - **SVG** (scalable, best quality)
   - **JPG/JPEG** (if no transparency needed)
   - **WEBP** (modern, efficient format)

## Supported File Names

The logo component will automatically look for:
- `logo.png` (primary)
- `logo.svg`
- `logo.jpg` or `logo.jpeg`
- `logo.webp`

## Recommended Logo Specifications

- **Size**: 200x200px to 400x400px (will be scaled automatically)
- **Format**: PNG with transparent background
- **Aspect Ratio**: Square (1:1) works best
- **Background**: Transparent (PNG) or matches site theme

## Current Status

If you don't have a logo file yet, the site will display the "SimexMafia" text with a gradient effect as a fallback.

Once you add your logo file, it will automatically appear in:
- Navigation bar (top left)
- Footer (top left)
